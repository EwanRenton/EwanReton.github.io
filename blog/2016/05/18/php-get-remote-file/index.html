PHP获取远程文件的几种方式

接上次的内容，今天结合代码来分析下PHP获取远程文件的几种方式。

  • fopen
    • 获取本地文件
    • 获取到的资源绑定到一个流上面
  • file_get_centents
    • 获取一些Get获得的数据，也可以POST，
    • 可以做断点续传
  • socket
  • curl
    • 可以保存cookie 并且在下一次发送出去
    • 做模拟登陆
